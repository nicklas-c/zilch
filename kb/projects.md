# Projects

## Guidelines for AI
This is an index of projects tracked in the knowledge base.  It contains brief reference information only — detailed logs and digests live in individual project folders under `./projects/`.

Maintain this index in alphabetical order by name.

### Fields

| Column | Optionality | Guidance |
|---|---|---|
| Name | Required | Human-readable project name |
| Slug | Required | The kebab-case folder name used in `./projects/` and in log tags |
| Aliases | Optional | Other names this project is known by, comma-separated |
| Summary | Required | One-line description of what the project is |
