# ZILCH-49114 — Fix team tagging on DataDog Dashboards

**Type:** Task
**Parent:** ZILCH-47400

---

## Background

The [Merchant team is defined in DataDog](https://app.datadoghq.eu/teams/9705f76c-2fa3-11ee-8ad7-da7ad0900005/unicorn) and appears to have the slug/handle `unicorn` and display name `merchant`.

Some of the team's dashboards are attributed to `unicorn` and others to `merchant` — meaning neither value returns the full set of dashboards when filtering by team.

- Unicorn dashboards: https://app.datadoghq.eu/dashboard/lists?q=team%3Aunicorn
- Merchant dashboards: https://app.datadoghq.eu/dashboard/lists?q=team%3Amerchant

## Description

Identify all merchant team dashboards, taking into consideration that some may not have a team field set at all. Ensure all dashboards are correctly attributed to the `merchant` team.
