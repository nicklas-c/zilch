# ZILCH-49117 — Rebuild fee-service dashboard

**Type:** Task
**Parent:** ZILCH-47400
**Depends on:** ZILCH-49115, ZILCH-49116

---

## Context

One DataDog dashboard currently exists for `fee-service`: https://app.datadoghq.eu/dashboard/sdj-m79-tkv/fee-service

This includes panels for things such as CPU usage and heap size, overlapping considerably with the [APM dashboard for the service](https://app.datadoghq.eu/apm/entity/service%3Afee-service).

Currently no functional metrics are included.

## Scope

Rebuild the fee-service dashboard:

- Remove panels for metrics that overlap with the APM dashboard — they are redundant.
- Add panels for agreed metrics for fee-service. (See [ZILCH-49115](https://payzilch.atlassian.net/browse/ZILCH-49115))
- Add a description to the dashboard.
