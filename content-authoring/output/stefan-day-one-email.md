### Projects to be Aware of
You're in the Merchant team, but you're our first Level-5 front end engineer, so you're not going to be staying in a Merchant lane: you'll want to be providing opinions and leadership across a number of initiatives.  It'll be useful to be aware of the following projects:

* **UI Component Test Pack**: This was originally a Merchant Team initiative and is now in use across the department, partly driven by the QA manager.  It's a front-end test pack using Maestro for automation and Pact for mocking.  It allows the interated front end to be tested without recourse to an actual back end.
* **New Web/SSR**: This is an initiative that Chris Walker kicked off.  You'll understand it better than I do, but from what I know, the idea is to help unify the React Native code for the mobile app with the React code for the customer website.  It also aims to adopt Server Side Rendering.
* **React Native New Architecture**: By the time you read this, this project should be behind us, but it has been a big initiative so you should know about it.  It was simply a drive to adopt the RN's new architecture and was largely an effort in finding and fixing bugs caused by the transition.  There were plenty and there may still be some lurking.

There are always some big strategic projects in play, and you should be aware of the ones we're currently working on.  I'll talk you through these in our 1:1.
